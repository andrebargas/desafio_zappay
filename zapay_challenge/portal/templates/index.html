Proposta

Elabore uma aplicação em Python que consuma a API da SpaceX e seja capaz de apresentar as seguintes informações:

  • Próximo Lançamento
  • Último Lançamento
  • Próximos Lançamentos
  • Lançamentos Passados

A aplicação deve oferecer a opção ao usuário de qual informação ele deseja visualizar.


Implementação

Desenvolvio com o framework de Python Django, com o auxilio de algumas bibliotécas como Requests, Pytest, etc.
